Ship-to Location - Additional Defaults

 

The additional codes portion of the ship-to location setup will let you setup options similar to those from the customer default data.

Follow these steps to complete this panel:

  1. If needed, enter a sales tax code and set the tax exempt status. Orders shipping to this location will use this taxing method as opposed to the main customer defaults.
  2. If needed, enter a default supply warehouse.
  3. If needed, enter the secure codes to allow all buyers at this location to see certain secure items.
  4. If needed, enter a default FOB, How to Ship, and/or Ship Via code to use in place of those in the customer defaults.
  5. Mark if this location should be excluded from Web.ec, or if Web.ec maximums should be ignored for this location.
