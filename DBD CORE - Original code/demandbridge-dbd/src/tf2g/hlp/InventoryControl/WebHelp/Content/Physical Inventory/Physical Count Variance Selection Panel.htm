Physical Count Variance Selection Panel

 

DB Distributor > Inventory Control > Physical Inventory > Physical Count Rep/Update (ICS P5)

 

Use the Physical Count Variance Selection Panel to print the Physical Count Variance Report, which shows the count quantities and resulting variances to update in the inventory records and General Ledger when the update command is executed. Follow these steps to complete the panel:

 

  1. Enter the item information on the Item tab.
  2. Enter the product information on the Product tab.
  3. Enter the location information on the Location tab.
  4. Click Output Options to review the printer configuration, making changes as necessary.
  5. Click View Report or Print Report.
  6. Click Exit.
