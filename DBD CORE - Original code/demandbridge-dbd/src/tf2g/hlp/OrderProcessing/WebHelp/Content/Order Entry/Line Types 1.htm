Line Types When Using a PO Code

 

When setting up a new line using a PO, you have 2 options.

  1. Select the Custom check box. This will tell the system that you want to order an item that is already setup as a custom item under this customer, or that you will be setting up a new item for the customer on-the-fly. This will set the Line Type field to "Manufactured Item".
    or
  2. Select one of the following line types:
  • Stock Item - used to order a generic stock inventory item from a vendor.
  • Message - used to add a message line to the vendor, customer, or both.
  • Special Charge - generally used to add a fee for services instead of a physical item. For example, an imprint charge, embroidery, mailing services, etc.
  • Non-Stock Item - used to create a temporary item that will not be saved in the custom or inventory item files.
