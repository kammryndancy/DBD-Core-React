Order Processing Parameters - Tab 4

 

DB Distributor > Order Processing > Setup > System > System Setup > Parameters (OPS S1)

 

For each parameter listed below, you will see the available options, and if applicable, a recommended setting.

 

Parameter Description
Set the GP adjustment to the cost for "C" type release?

If set to "yes", the cost of the item will be entered in as a GP Adjustment. Not used normally.

Customer number to exclude If the previous answer is set to "yes", you will have the option of restricting it to a particular customer code if needed.
Require cost codes during req and order entry? If set to "yes", user will be prompted to enter a valid cost code per line on the order.
Report finder's fee on a cash basis? If set to "yes", the finders fee report will only report the data based upon cash receipts versus time of invoicing.
Vendor/Product code finder's fees rate required? If set to "yes", a table can be entered by vendor and product code for the amount of the finders fee if it varies.
Print PO formats based upon the PO sufix? If set to "yes", there will be an option to enter the PO Codes to sort through for the particular printing scenarios. For example, PO Codes 0-9 print and fax or email, A-T - Print only, U-Z – Image only.
Commission code default based upon the product code?

Set to "yes" if you would like to pay a different commission based upon the product code of the item.
Implement supply item programming? Not is use. Leave blank.
Allow specs for stock items? If set to "yes", the generic stock items may have item specifications entered like a custom item.
Change item master art status to "P" when PO printed? If set to "yes", the "art status" flag on the custom item will be set to "P" for "proof" automatically when a PO is printed for the item.
Display the item notes in order entry? If set to "yes", when the item code is entered or selected, any item notes will be displayed for the operator.
Track "bundled" pricing for custom jobs? This would be for the price per unit including item, special charges, freight, and sales tax.
Adjust gross pricing on order booking? Should the gross profit on the order booking report take GP adjustments into account?
Display total freight on lots after selection? Uses a pro-rated freight amount from the freight field on the lot record. Can be used to charge to the customer inbound freight for items.
Include requisitions on the order booking report? If set to "yes", releases of distributor owned goods will be included on the order booking report.
Enable hidden warehouse feature? Setting this to "yes" will enable you to designate warehouses as being a "hidden" warehouse. Inventory in hidden warehouses will be distributor-owned, and can be left off of reports. This would normally be used if you are ordering larger amounts of an item to receive a lower cost, but not show this inventory as on-hand inventory to the customer.
No customer/vendor terms description & lookup/setup? If this is set to "yes", the description for the terms code will not show in order entry, and you will not be allowed to perform a lookup on the field.
Enable memo fields in item master on screens 2 and 8? Memo fields are located in the reorder and customer location area of the item code. Informational only. No printing is available.
