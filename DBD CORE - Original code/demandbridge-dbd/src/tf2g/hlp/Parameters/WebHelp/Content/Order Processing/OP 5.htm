Order Processing Parameters - Tab 5

 

DB Distributor > Order Processing > Setup > System > System Setup > Parameters (OPS S1)

 

For each parameter listed below, you will see the available options, and if applicable, a recommended setting.

 

Parameter Description
Enter sell price on stock purchase orders?

If set to "no", it will default to the item's base price.
Default "print on" field in Order Entry to Options are "S" for Sales Order Thank You Letter, "I" for Invoice, and "P" for Packing List. Enter in a combination of any of these three options. Using only "I" is the most common setting.
Make GP adjustment based on salesperson cost for direct ship inventory? Salesperson Cost field can be used to enter a standard cost for an item regardless of item cost from a vendor. If the actual cost is different, the difference be used to create a gross profit adjustment in order to affect the commission paid for the line.
Default PO Receiving accounting period to current AR accounting period? If set to "yes", this will cause the period in PO Receiving to default to the accounting period of your Accounts Receivable.
Display customer category code during requisition/release entry? Displays customer category code from the customer master file.
Change all warehouse location reorder points when item master changes?

If set to "yes", if a new reorder point is entered on the item master, all warehouse records will be updated to match that item.
Default the sell price on stock PO item's base price? When entering in Purchase Orders for stock, should the sell price default to the items base price?
Default stock item sell price in order entry to item master price? When selling stock items in Order Entry, should the sell price default to what is on its item master record?
Enable new order type field in order entry? Needs to stay defaulted to "yes'.
Display customer phone number on requisition/release header screen? If you would like to see the customer's phone number on the order header screen for releases and billable releases, set this to "yes".
Bill as shipped PO's booked when PO is entered? Entering "yes" will put this on the "bookings" at time of entry. "No" will put it on the "bookings" at time of release.
Book inventory with this customer code upon release If the previous parameter is set to "no", you can set if you would like "bill as shipped" inventory items to be booked under a specific customer code upon release.
Enable order shipment history feature? Set to "yes" to have prompt in order history to the shipment history of where a particular order shipped and in what quantities.
Enable location's requisition maximum exceeded warning? If using maximums, prompt the operator with a message when maximums have been exceeded.
Enable item comment display in order, requisition, and transfer entry? If an item has item comments, and this parameter is set to "yes", the comments will display in the order entry and inventory transaction programs when the item is selected.
Ask suspend packing list printing question in Order Entry and Req/Rel Entry? If you would like to have the ability to delay the printing of the packing list, you will want to set this to "yes".
Enable Suggestive Selling feature? New Suggestive Selling feature has been added to DB Distributor. Now during Order Entry you can suggest to the operator alternate items that can be ordered in conjunction with the item ordered or an item that can be substituted. You can alert the operator just when the item is out of stock or every time the item is ordered. Next, select Suggestive Selling Setup and indicate first the item and then the suggested item. You may also enter up to a 60-character comment that will display along with the suggested item.
Enable additional 4 lines of customer ship-to address? The Customer Ship-to Address Setup has been modified to provide for an additional 4 lines of 30 characters each. This additional address will print on the packing list and is inserted just after the ship-to name prints. Your packing list must be formatted to handle the additional 4 lines. After entering the normal address information in the Setup program, the bottom half of the screen will clear and provide you the ability to set up the additional lines of address.
