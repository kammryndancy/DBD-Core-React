Order Processing Parameters - Tab 7

 

DB Distributor > Order Processing > Setup > System > System Setup > Parameters (OPS S1)

 

For each parameter listed below, you will see the available options, and if applicable, a recommended setting.

 

Parameter Description
Print special shipping "ship to" description on the invoice?

If the shipping location is different than the billing location, this will cause the information for both locations print on the invoice.
Allow entry of more than one quote per OE order? If this is not turned on, you will be limited in order entry to only pull in information from one RFQ at a time.
Create PO numbers that do not update to the order book? This will give you the ability to order items that will not update to the order bookings file.
PO Numbers selected to bypass the order book update If the previous parameter is set to "yes", enter in the PO Codes that you would like to use that will not update to the order bookings file.
Send pack list to print file during PO receipt update if not printed yet? If at the time of order entry you selected to "suspend" packing list printing, setting this parameter to "yes" will cause the packing list to be printed at the time of the PO Receiving update if it has not yet been printed.
Enable repeat order feature? Set this to "yes" if you would like to have the option of creating orders based off of previous orders. Note: this must be set to "yes" if you are using the eCommerce toolkit.
Save all orders as repeat orders?

Options are "Yes" (which can be overridden), "All" (which cannot be overridden),

Display prompt in requisition lines entry to return to order header? Only applies to the CHUI version of the software. Allows the operator to return to the order header after entering lines.
Display existing notepad when entering existing order in Order Entry and Req/Rel Entry? If set to "yes", the order notepad will automatically show when calling up an existing order that has notes attached to it.
Format to use for PO Recv Carton Labels? If carton labels are generated during PO Receiving, enter in the carton label format that you would like to use.
Default new items to "Y" for lotted items in PO Recv? Should be set to "yes" if you use lotted inventory.
Recv dept must be unique, allow entry in order entry? If set to "yes", when a new ship-to location is setup, if a Receiving Department is entered, it must be a unique code.
Enable inventory costing feature? Set this to "yes" if you would like to use the Inventory Job Costing feature. Click here for more details.
Display message in req entry if item is below ROP? This can be used to notify an operator if an item is currently below its reorder point.
Default "ship from" warehouse in req entry to whse with oldest lot? If you would like to have the "ship from" warehouse default to the warehouse containing the oldest inventory during order entry, then set this to "yes".
Enable displaying of customer item notes? This will allow for the display of the notes in AR customer master, Custom item master setup, inventory master setup, Order entry, and RFQ Entry.
Use country from zip code file as billing dept during SJ update? The county field is from the zip code file based upon ship-to zip code. This can be placed into the billing department field during the Sales Journal Update if this parameter is set to "yes".
Packing list printing recognizes warehouse 1st digit? Set this to "yes" if you would like to have items with different ownership to print on different packing lists. For example, if you have items on an order from warehouses 0001 and 9001, would you want them on the same packing list, or separate.
Default "print on" field in requisition entry to Options are "S" for Sales Order Thank You Letter, "I" for Invoice, and "P" for Packing List. Enter in a combination of any of these three options. Using "IP" is the most common setting.
