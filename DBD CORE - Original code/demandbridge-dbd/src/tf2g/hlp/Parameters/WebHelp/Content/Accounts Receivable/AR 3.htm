Accounts Receivable Parameters - Screen 3

 

DB Distributor > Accounts Receivable > Setup > System > System Setup > Parameters (ARS S1)

 

For each parameter listed below, you will see the available options, and if applicable, a recommended setting.

 

Parameter Description
Do you have the commission accounting system?

If you will be using the system to keep track of commissions, set this to "yes". If you set this to "no", the system will not keep up with commission pending or payable.
Default commission codes for Custom Items, Customer Specific Inventory, Special Charges, Inventory Items, Non-stock Items, and Sales Orders to "D" locations There are many ways to determine what commission code should be used to calculate commissions during the invoicing process. One method would be to calculate commission based upon the type of item being sold. In these six fields you can specify which commission code should be used as a default. Leave these fields blank if you will be using one of the other commission methods.
Use SIC code to hold customer commission code? On the main customer setup record, there is an SIC field used to store a standard industrial classification code. This piece of information is rarely used. If this parameter is turned "on", then the field can be used to store a default commission code based upon the customer. This would override the default commission code set on the salesperson.
Should salesperson's GP include invoice GP adjustment? DB Distributor provides the logic to add house or overhead charges before calculating commissions. If you want the gross profit that is reported on the commission reports to be adjusted based upon the house charges that are added, check this box.
Update alternate sales tax file - AS3? This should only be turned "on" for companies operating in the state of Illinois.
Allow override of billing address during order entry? Check this box if you wish to allow your operators to override the billing address that defaults from the customer master file with a different address.
Ask immediate print question of speed bill invoices?

If you need to immediately print your speed bill invoices, you have three choices: Always, Never or Customer Specific. Note: Speed bill invoices are invoices where you are creating the order through invoice entry and immediately creating the invoice for it.
Ask if contact information should be accessed when setting up a new customer? If you are using the DB Distributor Contact Management System, you will probably set up prospects in this database prior to their becoming customers. You have the option, by checking this box, to automatically retrieve contact information when setting up a new customer. This eliminates redundant data entry of address information.
Should contact address be updated when customer's is changed? DB Distributor can automatically change the address of any linked contacts when the customer file is changed. Click this box to enable this feature. The options are: Always, Ask First, or Auto and Credit information.
Exclude freight GP from GP calculation at invoice time? There are instances when you may enter freight as a line item on your customer invoice. If you do, it would normally update your order history header with the additional gross profit. You have the option by checking this box to exclude this freight from gross profit.
Position for vendor designator in comm code? In accounts payable, you can enter in a 1 digit number in the vendor record to be substituted into the commission code. This field designates which position of the commission code the "vendor designator" will be substituted into. This essentially changes the commission code that is going to be used on order lines associated with a particular vendor. This can be used if you want to pay a different commission rate based upon the vendor, or more commonly, you are adding a house charge by vendor.
Apply once per vendor If you only want the commission substitution to occur once per vendor on an order, you will want to check this box "on".
