Contact Setup - Status Tab

 

DB Distributor > Contact Management > Setup > Contacts - Status (T/M T1)

 

This tab is used for showing the most recent status of the contact, and also assigning selection codes to them.

 

Follow these steps to complete the tab:

  1. The available selection code will be displayed in a box on the left. Double-click on each code that should be applied to this contact to move it to the box on the right. You can also use the arrow buttons to move codes back and forth between the lists.
  2. The last contacted information will automatically populate based upon actions that you take in the Notes tab. This can be manually changed if needed.
  3. The follow-up date information will also automatically populate based upon actions that you take in the Notes tab. This can be manually changed if needed.
  4. The barometer is usually set to describe how "hot" a prospect is, or how close they are to becoming a customer. This will be set between 0 and 9, and is based upon actions you take in the Notes tab. This can be manually changed if needed.
  5. The annual sales field can be used in one of two ways. You can use this to say what the annual sales are for the contact, or you can use this to project what your expected annual sales would be to them if they were a customer.

 

Continue to the Notes tab.
