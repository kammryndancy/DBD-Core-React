Vendor Reference Report

 

DB Distributor > Accounts Payable > Reports > Vendors > Vendor Reference (APR L1)

 

The Vendor Reference Report is used to print a report that shows a side-by-side comparison of your vendors in vendor code order, and in vendor lookup order.

 

Follow these steps to run this report:

  1. This report will run by vendor.
  2. Optionally, enter a range of vendors to narrow your search.
  3. Review the Output Options.
  4. Click Print Report.
