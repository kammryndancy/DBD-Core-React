Bank Reconciliation Setup

 

The setup of the Bank Reconciliation module consists simply of setting up the Bank Rec Parameters and Bank Accounts. Select a link to learn more about a particular item.

Automatic Disbursement

 

In addition to setting up these items, there is also a parameter in the Accounts Payable system that will need to be answered in order use Bank Reconciliation:

 

Is there automatic check reconciliation?

 

This parameter needs to be answered "yes".
