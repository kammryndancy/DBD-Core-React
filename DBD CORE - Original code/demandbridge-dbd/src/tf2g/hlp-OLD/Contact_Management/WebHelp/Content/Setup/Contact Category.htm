Contact Category

 

DB Distributor > Contact Management > Setup > Supporting Files > Contact Category (T/M S2)

 

When setting up a new contact, you will be required to assign them a contact category code. The categories that you setup might vary depending upon the list types you have setup. Normally here you will be setting up different business or industry types. Since this category (if assigned to a prospect) will be used if the contact is turned into a customer, you might want to include your different customer category codes in this list.

 

Follow these steps to complete the panel:

  1. Enter in a 3 character code, or use the lookup to view/edit existing category codes.
  2. Enter in a description for the category code.
  3. Press Save.
