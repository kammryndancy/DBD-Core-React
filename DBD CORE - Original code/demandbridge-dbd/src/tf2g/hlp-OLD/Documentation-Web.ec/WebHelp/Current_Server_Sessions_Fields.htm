EC Active Session Log dialog fields and settings

The EC Active Session Log dialog is used to view active server sessions. The EC Active Session Log dialog contains statistics for the following information:

  • Session ID

  • Buyer's ID Code

  • Buyer's Customer Name

  • Buyer's Location Name

  • Session Expired?

  • Session Valid Timeout

  • Login Date Time SQL

  • Last Activity SQL

  • Logoff SQL
