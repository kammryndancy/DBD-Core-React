Rebuilding Warehouse Group Lists

Rebuilding Web.ec sort files requires the following user actions.

  1. Launch the Web.ec module.

  2. Select the "Rebuild Whse Group Lists" operation from the System Setup/Utilities menu. The Update Item Warehouse Groups utility starts.

  3. The system will prompt you that it is about to rebuild the order status sort files for the Web.ec system.

  4. When prompted, select "Yes.

  5. Click the button to close the Update Item Warehouse Groups utility dialog.

STOP. You have completed this task.
