EC Cust Catalog Item Maintenance dialog fields and settings

Clicking the "Item Setup" item from the Product Catalog menu launches the EC Cust Catalog Item Maintenance dialog. The following table contains the field descriptions and the setting options and requirements of the EC Cust Catalog Maintenance dialog. After using this dialog, you must run the Rebuild WebEC Sorts and Publish Buyer Catalogs for web Buyers to see the new item setup. See the User Instructions topic for more information.
