Web.ec System Parameters

The E-Commerce Parameters item is used to configure the system-wide parameter settings used by Web.ec. for web and TopForm operations.

E-Commerce Parameters Dialog

Selecting the "E-Commerce Parameters" from the Setup menu launches the E-Commerce Parameters Maintenance dialog with all fields empty. The E-Commerce Parameters dialog is divided into four tabs.
