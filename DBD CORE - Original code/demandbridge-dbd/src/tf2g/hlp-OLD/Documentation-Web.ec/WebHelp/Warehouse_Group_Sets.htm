Warehouse Group Sets

The "Warehouse Group Sets" item is used to set up warehouse groups for customers, items, or generic warehousing . This is an optional feature available with the licensing of the Extended Logistics module.

Extended Warehouse Groups dialog

Selecting the "Warehouse Group Sets" item from the System Setup/Supporting File Setup menu launches the Extended Warehouse Groups dialog. The following figure shows an example of this dialog.
