Using the Catalog Category Change utility

Renaming and moving catalog categories requires the following user actions.

  1. Launch the Web.ec module.

  2. Select the "Category Rename/Delete" item from the Product Catalog menu. The Catalog Category Change utility dialog appears.

  3. Complete the settings in the dialog according the Dialog Fields and Settings for this topic.

  4. Click the button to commit the field entries.

  5. The system will automatically launch the Product Catalog Load Utility to load the catalog and the Publish Buyer Catalogs utility and publish the catalog.

  6. Click the button to close the Catalog Category Change utility dialog.

STOP. You have completed this task.
