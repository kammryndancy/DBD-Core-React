Vendor Ink Chart

 

DB Distributor > Order Processing > Setup > Supporting Files > Optional > Vendor Ink Chart (OPS O3)

 

Use the Vendor Ink Chart panel to enter in the ink colors associated with the PMS numbers for a specific vendor.

 

Follow these steps to complete this panel:

  1. Enter in the Vendor code.
  2. Enter in the PMS Number.
  3. Enter in the Ink Color associated with this PMS number for the vendor.
  4. Press Save.
