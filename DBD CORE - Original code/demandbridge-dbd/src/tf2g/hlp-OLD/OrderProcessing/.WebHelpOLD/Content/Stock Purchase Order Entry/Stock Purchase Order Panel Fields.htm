Stock Purchase Order Panel Fields

 

Field Description
Order Date The date of the order, typically today's date. Use the query button to change the date.
CSR Your Customer Service Representative code. Enter a CSR code or select one using the query button.
FOB User-defined. Optionally, select a free on board shipping method.
How to Ship User-defined. Optionally, select how to ship the order.
Ship VIA User-defined. Optionally, select a carrier.
