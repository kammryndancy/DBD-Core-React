Order Info Tab

 

DB Distributor > Order Processing > Purchase Orders > Processing > Purchase Order Entry > Stock Purchase Order > Purchase Order Entry > Purchase Order Setup > Order Info Tab (P/O P1 #1)

 

Use the Order Info Tab on the Purchase Order Setup Panel to define the order information for the order. Follow these steps to complete the tab:

 

  1. Select Print This P/O if you want to print the order information.
  2. Select the New/Repeat category from the drop-down list.
  3. Select Confirming if you are confirming a previous order.
  4. Enter the Gr. Cutoff Date or select it using the query button.
  5. Enter the quote in the Quote field.
  6. Enter the date or select it using the query button.
  7. Enter the terms or select it using the query button.
  8. Select the Other Tab.
