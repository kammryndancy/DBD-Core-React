RFQ Quick Entry

 

DB Distributor > Request for Quotation System > Processing > RFQ Quick Entry (RFQ A5)

 

The RFQ Quick Entry screen is used for making quick changes to existing RFQs. It is not used as a quick method for setting up new quotes.

 

Follow these steps to complete the panel:

  1. Enter in the RFQ number to make changes to. You can use the lookup button to view all of the currently open RFQs.
  2. If needed, mark the quote as being complete.
  3. If needed, change the CSR listed on the quote.
  4. If needed, enter in what the next action code should be on the quote.
  5. If needed, set a date for the next action code.
  6. If needed, edit the 3 comment lines to be printed on the customer quote letter.
  7. Press the Save button to update these changes to the RFQ.
