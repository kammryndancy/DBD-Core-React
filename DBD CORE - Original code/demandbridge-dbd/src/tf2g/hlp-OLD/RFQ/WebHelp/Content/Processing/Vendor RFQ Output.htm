Vendor RFQ Output

 

DB Distributor > Request for Quotation System > Processing > Vendor RFQ Output (RFQ A2)

 

The Vendor RFQ Output is used for sending your quotes to your vendors.

 

Follow these steps to print/fax/e-mail your quotes:

  1. The options field will normally be set to a "1". This is for the standard format of the RFQ. If you have setup different variations of the RFQ, you might have additional numbers to enter here.
  2. You can enter in a range of RFQ numbers, Line Numbers, and/or Vendors to control which RFQs will be sent out.
  3. Even if you will be faxing or e-mailing the quotes, leave the "output" field set to "printed requests".
  4. The "user defined spec format" field can be left blank, as it is not used with electronic forms.
  5. Your Output Options button should be defaulted to N0.
  6. Press the Print Report button to print/fax/e-mail your quotes to your vendors.
