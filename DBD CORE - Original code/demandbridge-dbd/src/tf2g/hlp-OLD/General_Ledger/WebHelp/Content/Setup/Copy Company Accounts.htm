Company Chart of Accounts Copy Utility

 

DB Distributor > General Ledger > Setup > System > System Setup > Copy Company Chart of Acct (GLS Z1)

 

This utility gives the option of copying all or a part of one company's chart of accounts to another company. This could be used if your company has multiple company codes.

 

Follow these steps to complete the panel:

  1. Enter the company code to be copied from.
  2. Enter the company code to be copied to.
  3. Designate if you would like for existing accounts to be removed. This will remove all accounts on the destination company.
  4. Designate if you would like for existing accounts to be overwritten if the same account number is to be copied in. This will leave the account numbers in tact on the destination company, and update any duplicate accounts.
  5. Enter in the starting and ending account numbers to copy from. To copy all accounts, enter in all zeros and all nines.
  6. Press Proceed to run the utility.
