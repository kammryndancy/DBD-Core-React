Source Journal Code Setup

 

The General Ledger System comes setup with the most used Journal Posting Codes.  See the table below for a list of codes and their descriptions.  Note that the Source Journal that are in Bold are those that would be entered via General Journal Entry.  You can create multiple codes to use when entering Journal Entries so that you can track the different types of entries.  

 

 

Code

Journal Description

CD

Cash Disbursement Journal

CR

Cash Receipts Journal

FC

Finance Charge Journal

IT

Inventory Transaction Journal

JE

General Journal

MC

Manual Check Disbursements Journal

MP

Manual Check Purchases Journal

P1

Physical Inventory Journal

PR

Payroll Journal

PU

Purchases Journal

SJ

Sales Journal

 

 

To setup additional Source Journal Codes, simply enter the code followed by its description.

 

Each Detail Transaction Register that prints, shows every posting that can be subsequently made to the General Ledger sequenced by Source Journal Code. You have the option to summarize this information before posting to the General Ledger by checking the appropriate box. For example, normally, the Finance Charge Journal would be summary posted, that is, you would not want to see each individual charge in your general ledger.  However, your purchases journal would be detail posted because you would want to see each vendor code, name and invoice details in your appropriate expense accounts.

Entering Budgets

 

Choose the Account Balance Option from the General Ledger Main Setup Folder.  Enter in the first account number for which you want to enter budget data.  Click on the Budget button.  Enter the budget amount for period one and press the enter key.

 

 

You will be given the option to repeat this entry for all twelve periods.  If more than half of the entries are the same, click the “Yes” button.
