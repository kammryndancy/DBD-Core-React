Reorder Projection

 

DB Distributor > Inventory Control > Reports > Reorder Projection (P/O R9)

 

Use the Inventory Reorder Projection Report to view a report that will show reorder points of your inventoried items, along with a current on hand/available amount, and average usages over user-defined periods of time.

 

Follow these steps to run this report:

  1. Select if you want to run the report by Item Code or Product Code.
  2. Optionally, you can enter a range to narrow your search. The criteria used to narrow your report will be determined by the sequence you chose to run the report by in step 1.
  3. Optionally, the remaining questions can be used to narrow your search, and to control what information is printed on the report.
  4. Review the Output Options.
  5. Click Print Report.
