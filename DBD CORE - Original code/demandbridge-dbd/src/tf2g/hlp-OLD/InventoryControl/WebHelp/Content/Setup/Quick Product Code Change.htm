Quick Product Code Change

 

DB Distributor > Inventory Control > Setup > Quick Product Code Change (I/C S5)

 

The Quick Product Code Change panel gives you the option of quickly changing the product code of an inventory item.

 

Follow these steps to complete this panel:

  1. Enter the customer code, or leave the field blank for a stock item.
  2. Enter the item code to be changed, or select it from the query.
  3. Modify the Product Code field.
